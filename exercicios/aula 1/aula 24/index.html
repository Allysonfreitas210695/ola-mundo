CSS abreviado com propriedade de fundo

Tente aumentar e diminuir o tamanho do seu navegador em todas as direções. Você vai perceber que o fundo vai se adaptar para que ele cubra todo o espaço útil da tela. Obtém esse resultado usando o parâmetro tamanho do plano de fundo com o valor capa ( cobrir ).

O resultado visual desse exercício é exatamente o mesmo do anterior, mas a configuração do CSS foi mais simplificada, usando atalhos. Dá uma visualização lá no arquivo style.cssdessa pasta.
